# AI Collaboration Prompts

The following are the prompts used to guide the AI assistant throughout the development of the AutoHub application.

### Prompt 1
"Act like a senior full-stack engineer building a production-ready startup MVP from scratch. First design the complete system architecture, then build the most minimal but scalable version possible. Include System architecture, File structure, Database schema, API endpoints, UI architecture, Production-ready code. Build it like a real startup that could scale to millions of users. my requirments TDD Kata: Car Dealership Inventory System Objective The goal of this kata is to design, build, and test a full-stack Car Dealership Inventory System."

### Prompt 2
"but i need to know that can i change the databases after we start building"

### Prompt 3
"explain me JWT, prisma orm and how it works and what are the requirmnts to use this in the current project"

### Prompt 4
"Now that the backend is fully tested and functional, scaffold a React frontend using Vite and TailwindCSS. Build a global AuthContext using the React Context API to manage JWT state, and create an Axios interceptor to handle authorization headers for protected routes."

### Prompt 5
"Use Git for version control. Structure your commits logically following the Conventional Commits specification, separating the core architecture setup, the TDD business logic implementation, and the frontend scaffolding into distinct commits."
